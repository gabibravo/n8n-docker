# Contributing to n8n Docker Setup

Este proyecto es un setup de n8n en Docker para facilitar despliegues.

## Cómo Contribuir

1. Fork este repositorio
2. Crea una rama: `git checkout -b feature/tu-feature`
3. Haz cambios y commit: `git commit -am 'Add feature'`
4. Push: `git push origin feature/tu-feature`
5. Abre un Pull Request

## Reportar Issues

Si encuentras problemas:
1. Describe el issue claramente
2. Incluye los logs relevantes
3. Especifica tu versión de Docker
4. Proporciona pasos para reproducir

## Estándares de Código

- Scripts bash deben ser POSIX-compliant
- Usar `set -e` para scripts críticos
- Documentar cambios significativos en README.md
